# 💪 健身记录 - Fitness Tracker

一个现代化的健身训练记录Web应用,帮助你记录每一次进步,保持训练动力!

## ✨ 功能特点

### 📝 训练记录
- **详细记录**: 记录每个动作的组数、次数和重量
- **身体部位分类**: 支持胸、背、腿、肩、手臂、核心、有氧等分类
- **照片记录**: 上传训练照片,记录身体变化
- **训练感受**: 添加个人备注和教练反馈
- **自动计时**: 自动记录训练时长

### 📊 数据统计
- **周报**: 每周训练总结,包括:
  - 训练频率和总时长
  - 身体部位训练分布
  - 与上周的进步对比
  - 本周照片集锦
  
- **月报**: 每月训练报告,包括:
  - 月度训练概览
  - 每周训练趋势图
  - 训练量进步曲线
  - TOP5常练动作
  - 月初月末对比照片

### 📤 分享导出
- **一键导出**: 将周报/月报导出为精美图片
- **朋友圈分享**: 适合分享到社交媒体

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

应用将在 `http://localhost:5173/` 启动

### 构建生产版本
```bash
npm run build
```

## 📱 使用指南

### 1. 添加训练记录
1. 点击右上角"新建训练"按钮
2. 选择训练日期(默认为今天)
3. 添加训练动作:
   - 选择身体部位
   - 输入动作名称(可从常见动作列表选择)
   - 添加每一组的次数和重量
4. 上传训练照片(可选)
5. 添加训练感受和教练反馈
6. 点击"保存训练"

### 2. 查看训练历史
- 点击底部导航栏的"首页"
- 在"最近训练"区域查看最近的记录
- 点击"查看全部"查看所有历史记录
- 可以编辑或删除任何记录

### 3. 查看周报
- 点击底部导航栏的"周报"
- 使用左右箭头切换不同周
- 点击"导出为图片"保存周报

### 4. 查看月报
- 点击底部导航栏的"月报"
- 使用左右箭头切换不同月份
- 点击"导出为图片"保存月报

## 🎨 技术栈

- **前端框架**: React + TypeScript
- **构建工具**: Vite
- **样式**: TailwindCSS (玻璃态设计)
- **图表**: Recharts
- **日期处理**: date-fns
- **图标**: Lucide React
- **数据存储**: LocalStorage (浏览器本地存储)
- **图片导出**: html2canvas

## 💾 数据存储

- 所有数据存储在浏览器的LocalStorage中
- 数据仅保存在本地,不会上传到服务器
- 清除浏览器数据会导致记录丢失,请定期导出备份
